# The Local AI

A clean starter layout for building and maintaining a local AI project.

## Repository layout

```text
.
├── .github/
│   └── workflows/
│       └── ci.yml
├── docs/
│   └── architecture.md
├── src/
│   └── README.md
├── tests/
│   └── README.md
├── .editorconfig
├── .gitignore
└── README.md
```

## Getting started

1. Clone the repository.
2. Read `docs/architecture.md`.
3. Place implementation code in `src/` and tests in `tests/`.
4. Keep CI green before merging.

## Project hygiene

- Keep docs and code in sync.
- Add tests for each new feature.
- Use conventional commit messages where possible.
